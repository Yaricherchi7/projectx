# Resend

## Programmatically send emails!

---

1. Go to https://resend.com/ and create an account

2. Choose any team name

You will be redirected to your dashboard.

3. Click **Add API Key** and copy the value to your `.env.local` file

4. In your `.env.local` file, set `TEST_EMAIL_ADDRESS` to the email address associated with your Resend account
