1. Go to https://edgestore.dev/ and create an account

You will be redirected to your dashboard.

3. Click **Create your first project** and choose any project name

4. Copy the keys to your `.env.local` file

## Done!

All the setup required to use Badget is now complete!

Run `pnpm run dev` to start the development server and navigate to `http://localhost:3000` to see your app in action.
