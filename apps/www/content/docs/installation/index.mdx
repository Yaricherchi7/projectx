---
title: Setup
description: Just some basic web development prerequisites!
---

1. You need Node.js installed on your machine. You can download it from [here](https://nodejs.org/en/download/).
2. `pnpm` is our package manager. You can install it by running `npm install -g pnpm`

Confirm correct installation by running `node -v` and `pnpm -v` in your terminal. You should see the version numbers, if installed correctly.

3. Clone the repository by running `git clone https://github.com/projectx-codehagen/Badget`
4. Install the dependencies by running `pnpm install`
5. Run `cp .env.example .env.local`. You will store your environment variables in this file.
